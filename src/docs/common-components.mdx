# BX UI Components

`bx-ui-components` is the shared component library for BX frontend projects. It provides reusable React UI components, shared styles, Storybook documentation, and component-owned translation strings.

## How Translations Work

Common component text is loaded from this package, not from the consuming app.

The component calls `useCommonComponentStrings()`, the hook reads the locale from the first URL segment, and it resolves that locale from the package locale folders. For example, `/ca-fr/sample` uses `ca-fr`.

The fallback order is exact locale, partial locale, and then `en`.

## Locale File Shape

Every package locale folder contains one file only:

```txt
locales/<locale>/common-components-strings.json
```

The `locales` folder is the source of truth. `src/i18n/locales.ts` only imports those locale files so the package can bundle them for React and Next.js apps.

The component library should not ship app namespaces such as `bx_v1.json`, `common.json`, or `course.find_course.json`.

```bash
npm run typecheck
npm run build
npm run pack:local
```

## Story Coverage

The grouped story sections are:
